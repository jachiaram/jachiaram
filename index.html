About Me

Hi I'm Jared Chiaramonte, a passionate senior-year computer science student interested in concepts in software engineering and cybersecurity. I recently worked at Northern Trust adding code coverage test cases, debugging web-sockets, and enhancing features on React.js micro front-ends. Last summer I worked on a software interface with the purpose of providing a tool to effective perform scalable artifact analysis on the Maven repository, with funding from the DAAD-rise program. I am constantly driven to keep learning new things in the technology space, from new technologies to different cybersecurity concepts.

Projects

MARIN (MAven Research INterface)

An interface focused on creating an accessible and scalable way to do research on artifacts on the Maven Central repository. MARIN contains an overarching implementation of the different modules in the interface, allowing for quick and repeated analysis runs to be performed.

RecipeEstimator

A Google Chrome extension for estimate costs to make different recipes. This extension enables easy estimation through webscraping and few API calls to retrieve local prices.
