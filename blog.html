Basic Buffer Overflows

Febrary 26, 2026

In this post I will introduce the topic of Buffer overflows. So to begin we will need to understand what a buffer is. A buffer is a finite region of memory that can be used to store data. In this post we are only discussing buffers that exist on the stack (i.e. those that are relative to the current frame of the program). Now that we have defined a buffer we can talk about overflowing buffers, this happens when you write past the end of the buffer. This is dangerous as the control flow of the program can be hijacked allowing hackers to gain access to files they shouldn't be able to or perform other malicious actions using the program.

Introduction to Pinyin and Tones

March 1, 2026

In this blog post I will discuss the basics of beginning to learn Mandarin Chinese, specifically pinyin. When you start learning Chinese usually the first thing you learn is pinyin which is a system used to romanize the different sounds in Mandarin. In pinyin there are different initials and finals, which when combined create different sounds. Each final also can be one of four different tone (1,2,3,4) high, rising, falling-rising, and falling.
